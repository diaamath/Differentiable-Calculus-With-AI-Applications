Section 4.3
Increasing and Decreasing

A function is increasing when its values grow as the input increases, and decreasing when its values become smaller.

If f'(x) > 0 → Increasing Function If f'(x) < 0 → Decreasing Function

In AI training, the model accuracy increases while the loss decreases.

تكون الدالة متزايدة عندما تزداد قيمها، ومتناقصة عندما تقل قيمها. وفي تدريب نماذج الذكاء الاصطناعي تزداد الدقة وتتناقص قيمة الخطأ.

Connection to Artificial Intelligence

Voice AI models learn to recognize speech by reducing prediction errors.

During training, the recognition accuracy increases while the loss function decreases.

تتعلم نماذج التعرف على الصوت من خلال تقليل الخطأ. حيث تزداد الدقة وتتناقص قيمة الخطأ أثناء التدريب. 


Voice Recognition Example

Audio Input = "hello"

Model Confidence = p

Loss Function:

Loss = (1 − p)²

Goal:

Increase p
Decrease Loss
